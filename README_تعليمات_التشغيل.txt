========================================================
  نظام نقطة المبيعات (POS System)
  تعليمات التثبيت والتشغيل
========================================================

المتطلبات:
----------
1. Visual Studio 2022 (مجاني - Community Edition)
   رابط التحميل: https://visualstudio.microsoft.com/

2. .NET 8 SDK (يأتي مع Visual Studio)

3. SQL Server LocalDB (يأتي مع Visual Studio تلقائياً)
   أو SQL Server Express (مجاني):
   https://www.microsoft.com/sql-server/sql-server-downloads

خطوات التشغيل:
---------------
1. افتح الملف: POSSystem.sln في Visual Studio

2. قم ببناء المشروع:
   - من القائمة العلوية: Build > Build Solution
   - أو اضغط: Ctrl + Shift + B

3. شغّل التطبيق:
   - اضغط F5 أو زر Run الأخضر

4. عند أول تشغيل:
   - سيتصل البرنامج بقاعدة البيانات تلقائياً
   - سيُنشئ الجداول والبيانات الأولية

بيانات الدخول الافتراضية:
---------------------------
المدير:  PIN = 1234
الكاشير: PIN = 0000

تغيير قاعدة البيانات:
----------------------
إذا أردت الاتصال بـ SQL Server العادي (وليس LocalDB):
افتح ملف: ConnectionString.txt وعدّل سلسلة الاتصال:

مثال لـ SQL Server Express:
Server=.\SQLEXPRESS;Database=POSSystem;Trusted_Connection=True;TrustServerCertificate=True;

مثال مع اسم المستخدم وكلمة المرور:
Server=localhost;Database=POSSystem;User Id=sa;Password=YourPassword;TrustServerCertificate=True;

مميزات النظام:
--------------
✅ شاشة كاشير مع شبكة أصناف ملونة
✅ تصنيف الأصناف بألوان مميزة
✅ البحث بالاسم، الباركود، رقم الصنف
✅ إدارة الفواتير (إنشاء، تعديل، إلغاء)
✅ طرق دفع متعددة (نقدي، شبكة، بطاقة، تحويل، محفظة، مختلط)
✅ إدارة الأصناف والفئات
✅ إدارة العملاء
✅ إدارة الموظفين وصلاحياتهم
✅ تقارير المبيعات
✅ طباعة الفواتير
✅ إحصائيات اليوم
✅ تسجيل الدخول بـ PIN

اختصارات لوحة المفاتيح:
-------------------------
F2  - البحث عن صنف
F5  - فاتورة جديدة
F12 - إتمام الدفع
Del - حذف صنف من الفاتورة

للدعم الفني والتطوير:
----------------------
يمكن تعديل الكود المصدري بالكامل في Visual Studio
